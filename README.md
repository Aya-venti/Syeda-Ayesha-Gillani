**Portfolio: Frontend Developer**
A modern, professional portfolio website showcasing frontend development skills with a focus on clean UI, performance, and security-conscious design.

**Overview**
This portfolio is built for a frontend developer with a background in security and machine learning. It highlights technical expertise, project work, and a clear call-to-action for internship opportunities in Summer 2026.

The design balances a dark, sophisticated aesthetic with subtle amber and teal accents, creating a memorable brand identity that feels both technical and approachable.

**Features**
Hero Section with dynamic typing effect

Interactive Project Filter (web, security, ML categories)

Technical Stack Display with pill-style badges

About Section with key metrics

Terminal-style Card showing developer profile

Responsive Design (Bootstrap 5)

Smooth Scroll Navigation

Active Nav Link Highlighting on scroll

Accessibility Features (prefers-reduced-motion support)

**Technologies Used**
HTML5 — Semantic markup

CSS3: Custom properties, flexbox, grid, animations

Bootstrap 5.3.3: Layout, components, responsiveness

Bootstrap Icons 1.11.3: Iconography

Google Fonts: Space Grotesk, Inter, JetBrains Mono

Vanilla JavaScript: Typing effect, filtering, scroll behavior
